# all terminal command
* touch <filename>
* mkdir <foldername>
* cd <foldername>
* cp <source> <destination>
* cp -r <source> <destination> //incase of folder
* rm <filname>
* rm -r -f  <foldername>
* rm -rf  <foldername>
* mv <source> <destination>

## selection 
* parent directory ..
* own directory .
* all *

## flag
* r - recursive
* f - force
